# 타임시트 (Time Tracker)

시간별 활동을 계획하고 기록할 수 있는 웹 기반 타임시트 애플리케이션입니다.

## 주요 기능

### ⏰ 시간 관리
- **24시간 시간대**: 4시부터 3시까지 모든 시간대 표시
- **날짜 선택**: 달력을 통한 날짜 선택 및 어제/오늘/내일 네비게이션 버튼
- **계획 vs 실제**: 각 시간대별로 계획된 활동과 실제 활동을 분리하여 기록

### 📝 데이터 입력 및 관리
- **드래그 선택**: 클릭 후 드래그하여 복수의 시간대 선택
- **단일 선택**: 개별 필드 클릭으로 선택/해제
- **독립적 선택**: 계획된 활동과 실제 활동 열이 각각 독립적으로 선택 가능

### 🔗 셀 병합
- **복수 셀 병합**: 선택된 시간대들을 하나의 큰 입력 필드로 병합
- **병합 버튼**: 복수 선택 시 선택 영역 중심에 병합 버튼 자동 표시
- **엑셀 스타일**: 병합된 셀은 전체 크기를 유지하면서 하나의 입력 칸으로 표시

### 💾 자동 저장
- **로컬스토리지**: 브라우저 로컬스토리지를 통한 자동 데이터 저장
- **날짜별 저장**: 각 날짜별로 독립적인 데이터 관리
- **자동저장**: 입력 후 2초 지연으로 자동 저장
- **수동 저장/불러오기**: 저장, 불러오기, 초기화 버튼 제공

### 📊 자동 계산
- **시간 합계**: 계획된 활동과 실제 활동의 총 시간 자동 계산
- **실시간 업데이트**: 입력과 동시에 합계 자동 갱신

## 사용법

### 기본 사용법
1. **날짜 선택**: 상단의 날짜 선택기나 네비게이션 버튼으로 원하는 날짜 선택
2. **활동 입력**: 각 시간대의 계획된 활동과 실제 활동 입력
3. **자동 저장**: 입력 내용이 자동으로 저장됨

### 선택 및 병합
1. **단일 선택**: 입력 필드를 클릭하여 선택/해제
2. **범위 선택**: 입력 필드를 클릭한 채로 다른 필드까지 드래그
3. **병합**: 복수 선택된 상태에서 나타나는 "병합" 버튼 클릭
4. **선택 해제**: ESC 키 또는 다른 영역 클릭

### 키보드 단축키
- **ESC**: 모든 선택 해제
- **Ctrl + 클릭**: 기존 선택을 유지하면서 추가 선택

## 기술 스택

- **HTML5**: 기본 구조
- **CSS3**: 스타일링 및 반응형 디자인
- **Vanilla JavaScript**: 모든 기능 구현
- **LocalStorage**: 데이터 저장

## 브라우저 지원

- Chrome, Firefox, Safari, Edge 등 모든 모던 브라우저 지원
- 반응형 디자인으로 모바일 기기에서도 사용 가능

## 시작하기

1. 프로젝트 파일을 다운로드하거나 클론합니다
2. `index.html` 파일을 웹 브라우저에서 엽니다
3. 바로 사용 시작!

별도의 서버 설치나 설정이 필요하지 않습니다.

## 주의사항

- 데이터는 브라우저의 로컬스토리지에 저장되므로 브라우저를 완전히 초기화하면 데이터가 사라질 수 있습니다
- 다른 기기에서 동일한 데이터를 사용하려면 수동으로 내보내기/가져오기 기능을 사용하세요
