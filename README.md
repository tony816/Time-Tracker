# 타임시트 (Time Tracker)

실시간 타이머 기능이 포함된 고급 시간 관리 및 활동 기록 애플리케이션입니다.

## 주요 기능

### ⏰ 시간 관리
- **24시간 시간대**: 4시부터 3시까지 모든 시간대 표시
- **3단 레이아웃**: 계획된 활동 | 시간 컨트롤 | 실제 활동
- **날짜 선택**: 달력을 통한 날짜 선택 및 어제/오늘/내일 네비게이션 버튼
- **스마트 계획**: 좌측 열에서만 스케줄 입력 가능

### ⏱️ 실시간 타이머 시스템
- **시간 기반 활성화**: 현재 시간과 계획된 활동이 있을 때만 타이머 버튼 활성화
- **개별 타이머 컨트롤**: 각 시간대별 독립적인 ▶️/⏸️/⏹️ 버튼
- **단일 타이머 정책**: 한 번에 하나의 타이머만 실행 가능
- **실시간 표시**: 타이머 실행 중 실시간 경과 시간 표시 (00:15:32 형태)
- **자동 기록**: 타이머 정지 시 자동으로 실제 활동에 기록
- **시각적 피드백**: 타이머 실행 중 레드 테마 배경으로 강조

### 📝 고급 활동 기록
- **상세 기록 모달**: 📝 버튼을 통한 포괄적 활동 기록
  - 활동 제목
  - 상세 내용 
  - 결과/성과
- **자동 연동**: 상세 기록의 제목이 실제 활동 필드에 자동 반영
- **구조화된 데이터**: 체계적인 활동 데이터 관리

### 🔗 동기화 셀 병합
- **3열 동기화 병합**: 좌측 계획 열 병합 시 시간 열과 실제 활동 열 자동 병합
- **시간 범위 표시**: 병합된 시간 열에 "12-13" 형태 시간 범위 표시
- **통합 인터페이스**: 병합된 실제 활동 열의 입력 필드와 버튼 1개로 통합
- **스마트 병합**: 기존 디자인을 유지하면서 기능적 통합

### 📊 성과 분석
- **실행율 계산**: 계획 대비 실제 실행 비율 (%)
- **타이머 사용량**: 총 타이머 사용 시간 (시간/분/초)
- **색상 코딩**: 성과에 따른 시각적 표시 (좋음/보통/개선필요)
- **실시간 업데이트**: 활동 완료와 동시에 분석 데이터 갱신

### 💾 고급 데이터 관리
- **로컬스토리지**: 브라우저 로컬스토리지를 통한 자동 데이터 저장
- **날짜별 저장**: 각 날짜별로 독립적인 데이터 관리
- **자동저장**: 입력 후 2초 지연으로 자동 저장
- **하위 호환성**: 기존 데이터와의 완벽한 호환성 보장

## 사용법

### 기본 사용법
1. **날짜 선택**: 상단의 날짜 선택기나 네비게이션 버튼으로 원하는 날짜 선택
2. **활동 입력**: 각 시간대의 계획된 활동과 실제 활동 입력
3. **자동 저장**: 입력 내용이 자동으로 저장됨

### 선택 및 병합
1. **단일 선택**: 입력 필드를 클릭하여 선택/해제
2. **범위 선택**: 입력 필드를 클릭한 채로 다른 필드까지 드래그
3. **병합**: 복수 선택된 상태에서 나타나는 "병합" 버튼 클릭
4. **선택 해제**: ESC 키 또는 다른 영역 클릭

### 키보드 단축키
- **ESC**: 모든 선택 해제
- **Ctrl + 클릭**: 기존 선택을 유지하면서 추가 선택

## 주의사항

- 데이터는 브라우저의 로컬스토리지에 저장되므로 브라우저를 완전히 초기화하면 데이터가 사라질 수 있습니다
- 다른 기기에서 동일한 데이터를 사용하려면 수동으로 내보내기/가져오기 기능을 사용하세요

## 데이터 내보내기/가져오기

- 내보내기: 화면 하단의 `내보내기` 버튼을 클릭하면 현재 브라우저의 모든 날짜(`timesheet_YYYY-MM-DD`) 데이터를 JSON 파일로 저장합니다.
- 가져오기: `가져오기` 버튼을 클릭해 JSON 파일을 선택하면 데이터를 불러옵니다.
  - 동일 날짜 데이터가 존재하면 덮어쓰기 여부를 확인합니다(취소 시 중복 날짜는 건너뜀).
  - 지원 포맷
    - 전체 내보내기 포맷: `{ version, exportedAt, dates: { 'YYYY-MM-DD': { date, timeSlots, mergedFields } } }`
    - 단일 날짜 포맷: `{ date, timeSlots, mergedFields }`
    - 배열 포맷: `[ { date, timeSlots, mergedFields }, ... ]`
